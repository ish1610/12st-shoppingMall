## 프로젝트 소개
* 쇼핑 플랫폼 12st은 상품 구매와 관리자에 의한 상품 관련, 고객 관련 정보 수정이 가능합니다.
* React.js를 사용하여 플랫폼을 구축하였습니다.
* 현재 플랫폼은 사용자, 관리자의 기능으로 구성되어있지만, 세부적으로 사용자, 판매처 관리자, 플랫폼 관리자로 부분하여 구축하고 싶습니다.
